# Dashboard RH em Power BI
Dashboard interativo desenvolvido para análise de indicadores de Recursos Humanos.

## Funcionalidades
- Monitoramento de contratações e desligamentos
- Cálculo de turnover
- Distribuição de colaboradores por área e cargo
- Análise salarial por cidade
- Indicadores de diversidade de gênero

## Tecnologias utilizadas
- Power BI
- Excel
- DAX

## Objetivo
Facilitar o acompanhamento de métricas estratégicas de RH através de visualizações claras e interativas.
